WordPress 2.6

By Neil De Souza
August 3, 2008

I've been test running WordPress 2.6 for a week now and all looks good. Codenamed 'Tyner', 2.6 visually is different from my previous 2.3.2.

A few changes that are really cool are in the admin interface include theme previews without the need for applying it on your blog, tracking revisions to blog entries, and an awesome development on the plugins page - automatic plugin upgrades. lol. Kewl.

Along with this I also recently tried out WordPress Automatic Upgrade by Keith Dsouza. This plugin does what it says - automatically upgrades your web blog to the latest version in a step by step process (or manually if you like).
